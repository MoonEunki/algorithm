풀이를 떠올리는것은 쉽지만 코드로 구현하는게 까다로운 문제들.

# 상하좌우

```javascript
function solution(n, arr) {
  let x = 1;
  let y = 1;
  let dx = [0, 0, -1, 1];
  let dy = [-1, 1, 0, 0];

  for (const move of arr) {
    if (move === 'L') {
      mx = x + dx[0];
      my = y + dy[0];
    }
    if (move === 'R') {
      mx = x + dx[1];
      my = y + dy[1];
    }
    if (move === 'U') {
      mx = x + dx[2];
      my = y + dy[2];
    }
    if (move === 'D') {
      mx = x + dx[3];
      my = y + dy[3];
    }

    if (mx >= 1 && mx <= n && my >= 1 && my <= n) {
      x = mx;
      y = my;
    }
  }

  return [x, y];
}

console.log(solution(5, ['R', 'R', 'R', 'U', 'D', 'D']));
```

난이도가 낮은 간단한 문제였다 약간 반복되는코드를 좀더 줄인다면 더 좋을듯.
